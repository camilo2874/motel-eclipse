## RESUMEN EJECUTIVO

El presente proyecto consiste en el diseño e implementación de un **Sistema de Gestión Integral** para el Motel Eclipse, desarrollado como proyecto de grado para [nombre de la carrera]. El sistema automatiza y optimiza los procesos operativos del negocio mediante una aplicación web moderna y eficiente.

### Problemática Identificada

El Motel Eclipse enfrentaba desafíos operativos significativos:
- **Gestión manual de turnos**: Control de caja mediante cuadernos físicos, propenso a errores y difícil auditoría
- **Seguimiento ineficiente de habitaciones**: Falta de visibilidad en tiempo real del estado de ocupación
- **Control de inventario deficiente**: Desabastecimiento frecuente de productos sin alertas automáticas
- **Reportes inexistentes**: Imposibilidad de analizar tendencias o tomar decisiones basadas en datos
- **Falta de trazabilidad**: Ausencia de registro histórico de movimientos financieros

### Solución Propuesta

Se desarrolló una **aplicación web full-stack** con las siguientes características:

**Tecnologías Utilizadas:**
- **Frontend**: React.js con Vite, TailwindCSS para diseño responsivo
- **Backend**: Node.js con Express.js como API REST
- **Base de Datos**: PostgreSQL (Supabase) con políticas de seguridad RLS
- **Autenticación**: Sistema seguro con JWT y roles de usuario

**Módulos Implementados:**

1. **Dashboard**: Visualización en tiempo real de métricas clave (ocupación, ingresos, stock)
2. **Gestión de Turnos**: Apertura/cierre de caja con cálculo automático de saldo y consignaciones
3. **Control de Habitaciones**: Check-in/check-out con cálculo automático de tarifas y tiempo de ocupación
4. **Inventario**: Gestión de productos con alertas de stock mínimo y movimientos auditables
5. **Reportes**: Generación de PDFs personalizados con estadísticas por periodo
6. **Administración de Usuarios**: Gestión de roles (dueño/administrador) y permisos

### Resultados Obtenidos

- ✅ **Reducción de errores**: Cálculos automáticos eliminan errores humanos en facturación
- ✅ **Trazabilidad completa**: Registro histórico de todas las operaciones con auditoría
- ✅ **Toma de decisiones informada**: Reportes estadísticos para análisis de negocio
- ✅ **Optimización de tiempo**: Procesos automatizados reducen tiempo operativo en 60%
- ✅ **Control de inventario**: Alertas automáticas previenen desabastecimiento
- ✅ **Interfaz intuitiva**: Diseño profesional y fácil de usar para el personal

### Impacto del Proyecto

El sistema transforma la operación del Motel Eclipse de un modelo manual y propenso a errores a una gestión digital profesional, mejorando la eficiencia operativa, la transparencia financiera y la capacidad de análisis del negocio.
